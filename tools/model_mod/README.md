## [ONNX](./onnx_mod.py)
Usage:
```bash
./onnx_mod.py --help
usage: onnx_mod.py [-h] -m MODEL [-o OUTPUT] [--debug]

options:
  -h, --help           show this help message and exit
  -m, --model MODEL    Model to create data for

```
